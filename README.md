# ERC20-TST
Ethereum ERC-20 Test Standard Token (TST)

For testing of ERC-20 token apps. Create free TST by calling `showMeTheMoney(address _to, uint256 _value)`.

Available at:

- Ropsten testnet: 0xC53df728BED60dE8e4b35beA7Abd5aB08f33075d
